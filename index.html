Estrutura de uma Página

HTML: Indica que qualquer coisa dentro dela é código HTML;

HEAD: Possui dados da página e não serão exibidos na janela principal;

TITLE: É o título da página que será mostrado no navegador;

BODY: Contempla o que será mostrado na janela principal do browser.

Estrutura HTML:

  1. HTML é uma linguagem de marcação (estruturação) de texto usada em páginas web;
  2. Códigos são inseridos nas palavras que queremos que apareçam na página desenvolvida;
  3. Esses códigos são chamados de TAGs: onde geralmente há a de abertura e a de fechamento;
  4. O HTML informa ao browser o que deverá ser feito com os elementos que estão entre as TAGs.

Tipos de Marcação de Texto

Marcação Estrutural: Impacta na estrutura (apresentação) da página visualmente;

Marcação Semântica: Não afeta a estrutura da página visualmente, mas adiciona informações extras.

Trabalhando com Títulos

Existem 6 níveis de títulos (tamanhos) pré definidos para usarmos em nossas páginas web;

Os níveis variam de <h1> e <h6>, sendo o <h1> o maior deles e o <h6> o menor;

O tamanho da fonte pode variar um pouco de browser para browser.

Adicionando Parágrafos

  1. Usar parágrafos é importante para fluidez do texto;
  2. Cada parágrafo deve estar entre as TAGs: <p> e </p>;
  3. O browser por padrão os exibe com espaço entre eles.

Organizando Melhor as Ideias

  1. O HTML possibilita criar listas para organizar as informações;
  2. Imagine, por exemplo, o passo a passo de uma receita de bolo;
  3. Ou até mesmo um checklist para fazer as compras no mercado;
  4. Em ambos os casos é necessário estruturar as informações.

Lista Ordenadas

  1. Os itens dessas listas são númerados;
  2. A sua criação se dá com a tag: <ol>;
  3. Cada item da lista é um elemento: <li>;
  4. O atributo type.

Lista Não Ordenadas

  • Os itens da lista recebem marcadores;
  • É criada através do uso da tag: <ul>;
  • Cada item da lista é expresso por: <li>;
  • Também pode usar o atributo type.

Lista Aninhadas

  1. É possível colocar uma lista dentro de outra;
  2. Basta adicionar a lista dentro de um <li>
  3. Assim, o elemento se tornará uma lista;
  4. Haverá um recuo diferente da lista pai.

Trabalhando com Links

  1. O uso de links é considerado o principal recurso formador da web que conhecemos;
  2. É através deles que conseguimos navegar entre as milhões de páginas existentes na web;
  3. Os links podem ser criados para vários propósitos, por exemplo, interligar um site a outro.

Abrindo o link em uma nova janela!

  1. As vezes não queremos que o usuário saia da nossa página ao abrir um link para outro site;
  2. Ele não será afetado pela abertura da nova janela;
  3. Esse comportamento é implementado através do atributo atributo target junto com o valor valor_blank.

Link entre Páginas

  1. Interliga as páginas de um mesmo site;
  2. Deve-se indicar o nome do arquivo destino;
  3. Para isso é usado URLs relativas;
  4. São URLs abreviadas os URLs absolutos.

Links de E-mail

  1. Inicia o programa de e-mail do usuário para enviar uma mensagem;
  2. O valor do atributo href deverá conter:"mailton:exemplo@gmail.com";
  3. Esse tipo de link se parece com os demais com exceção da abertura do gerenciador de e-mail.

Dicas sobre Imagens

  1. As imagens podem ter direitos autorais sempre confira;
  2. Crie uma pasta em seu site para mantê-las organizadas;
  3. Use os formatos mais comuns/tradicionais: jpeg, gif ou png;
  4. A maioria das telas trabalham no máximo a 72 pixels por polegada.
  1. A imagem deve estar dentro de uma TAG do tipo figure;
  2. A legenda também deve estar dentro da mesma TAG;
  3. A legenda é definida com base em uma TAG figcaption.

Trabalhando com Tabelas

  1. Elas permitem mostrar dados em forma de grid (linhas e colunas);
  2. A tag <table> é usada para indicar a definição da tabela;
  3. Cada linha ou registro da tabela é representado pela TAG: <tr>;
  4. Já as suas células/colunas são expressas pela TAG: <td>;

Cabeçalho das Tabelas

  1. Para identificar o cabeçalho de uma tabela usamos a TAG: <th>;
  2. Podemos usá-la para identificar linhas ou colunas ao invés de <td>;
  3. Melhora a acessibilidade de sua página para leitores de tela.

Mesclando Colunas da Tabela

  1. Assim como no Excel é possível mesclar as colunas de uma tabela;
  2. Para isso é utilizado o atributo colspan dentro de uma célula;
  3. O valor de colspan deverá ser o tamanho da célula desejada;
  4. O valor será sempre menor ou igual ao nº de colunas da tabela.

Mesclando Linhas da Tabela

  1. Enquanto o colspan mescla colunas, o rowspan mescla linhas;
  2. O seu valor refere-se quantidade de linhas que serão mescladas;
  3. Esse valor deve ser menor ou igual ao número de linhas da tabela;
  4. O rowspan também é usado como atributo nas TAGs: <th> ou <td>.
